# メディアストリーミング録画 API

## 録画を開始する

### 基本情報

APIパス
: /gotapi/mediastream_recording/record

HTTPメソッド
: POST

### リクエスト

#### HTTPヘッダー

|ヘッダーの種類|内容|
|----------------|----------------|
|Content-Type|application/json|

#### パラメータ

|キー|必須|型|場所|説明|
|-----|-----|-----|-----|-----|
|serviceId|✓|文字列|クエリー|サービスID|
|accessToken|✓|文字列|クエリー|アクセストークン|

### レスポンス

#### HTTPレスポンス

|HTTPステータスコード|内容|
|-----|-----|
|200||

エラーについては [エラーについて](./error.md) を参照してください。


#### レスポンス内容

|キー|型|
|-----|-----|
|result|数値|
|product|文字列|
|version|文字列|

### リクエスト

|ヘッダーの種類|内容|
|----------------|----------------|
|Content-Type|application/json|

#### パラメータ

|キー|必須|型|場所|説明|
