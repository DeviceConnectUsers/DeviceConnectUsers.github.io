# ファイルの変更通知

## ファイル監視


事前に ファイルの変更を監視する Web APIを実行しておく必要があります。

```
{
    "attribute": "onwatchfile", 
    "file": {
        "curr": "2016-07-27T06:50:24+0900", 
        "path": "test"
    }, 
    "profile": "file_descriptor", 
    "serviceId": "Host.e87e3213b730843a437ff6c676899df0.localhost.deviceconnect.org", 
    "sessionKey": "vwt81tt9"
}
```
