電力メーター API

使用電力量を取得

センサーの取得した瞬間電力量を取得します。

基本情報

APIパス : /gotapi/powermeter/instantaneouspowervalue

HTTPメソッド : GET

リクエスト

HTTPヘッダー

ヘッダーの種類 内容
Content-Type application/json

パラメータ

キー 必須 型 場所 説明
serviceId ✓ 文字列 クエリー サービスIDを指定します
accessToken ✓ 文字列 クエリー 取得したアクセストークン

レスポンス

HTTPレスポンス

HTTPステータスコード 内容
200 HTTP経由でアクセスするファイル情報

エラーについては エラーについて を参照してください。

レスポンス内容

キー 型
result 数値
instantaneouspowervalue 数値
product 文字列
version 文字列
